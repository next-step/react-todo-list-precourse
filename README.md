# react-todo-list-precourse

## 구현 계획

1. 기초 틀 만들기
    1) 입력 칸
    2) 할일 목록 칸
        - 체크버튼
        - 삭제버튼
    3) 옵션
        - 아이템 개수 확인
        - 필터링
        - 모두 지우기
2. 기능 구현
    1) Enter키, 추가 버튼으로 추가
    2) 삭제버튼, 모두지우기로 삭제
    3) 완료 상태 전환
    4) 필터링 기능
3. 예외처리
    1) 입력 없을 때 추가하지 않게
    2) 새로고침해도 데이터 유지
4. UI
    1) 예쁘고 깔끔하게 만들기 (TodoMVC 참고)
