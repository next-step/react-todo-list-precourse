# react-todo-list-precourse

## 기능 정리

### 필수 기능

- [참고](https://todomvc.com/examples/react/dist/)
- [x] 할 일 목록 뷰
- [x] 할 일 추가
  - input에 입력한 후 Enter나 추가 버튼을 이용해 목록에 추가
  - 아무것도 입력되 있지 않은 경우 추가 불가
- [x] 할 일 삭제

### 추가 기능

- [ ] 할 일 필터링(진행 중, 완료 된 일, 전체)
- [ ] 해야 할 일의 총 개수 출력
- [ ] 새로고침 시에도 유지

## code convention

- 들여쓰기 depth 2 이하.
- 함수 길이 15라인 이하.
